隐私政策

最后更新日期：2026年2月5日

欢迎使用 AI Prompt Helper（以下简称“本扩展”）。我们需要向您说明我们如何处理您的数据。简而言之：我们不收集您的任何个人信息，您的数据完全存储在您的本地设备上。

1. 数据收集与存储

本扩展承诺不收集、不传输、不存储您的任何个人信息或使用数据到任何外部服务器。

  • 本地存储：您创建的所有提示词（Prompts）和设置选项均存储在您浏览器的 chrome.storage.local 中。这些数据仅保留在您的设备上，除非您主动将其导出或卸载本扩展。
  • 无云端同步：本扩展目前没有云端同步功能，因此我们无法访问您的任何数据。

2. 权限使用说明

为了实现功能，本扩展需要申请以下权限，我们仅在必要时使用这些权限：

  • storage / unlimitedStorage：用于在您的本地设备上保存您的提示词库和偏好设置。
  • activeTab / scripting：用于在您当前浏览的 AI 网页（如 ChatGPT, Claude 等）中自动填充您选择的提示词。我们仅在您主动点击扩展界面或使用快捷键时才会与页面交互。
  • contextMenus：用于在网页上添加右键菜单，方便您将选中的文本快速保存为新的提示词。

3. 第三方服务

本扩展纯本地运行，不包含任何第三方分析工具（如 Google Analytics）、广告 SDK 或追踪代码。

4. 数据安全

由于您的数据存储在本地，其安全性依赖于您设备的安全性。请勿 in 公共设备上保存敏感信息。

5. 政策更新

如果我们的隐私政策发生重大变更，我们将通过扩展更新说明或本页面通知您。由于本扩展不收集用户信息，我们无法通过电子邮件通知您。

6. 联系我们

如果您对本隐私政策有任何疑问，或者在使用过程中遇到问题，请通过 GitHub Issues 或应用商店评论区与我们联系。

Privacy Policy

Last Updated: February 5, 2026

Welcome to AI Prompt Helper (hereinafter referred to as "the Extension"). We want to be transparent about how we handle your data. In short: We do not collect any of your personal information; your data is stored entirely on your local device.

1. Data Collection and Storage

The Extension commits to not collecting, transmitting, or storing any of your personal information or usage data on any external servers.

  • Local Storage: All prompts and settings you create are stored in your browser's chrome.storage.local. This data remains solely on your device unless you manually export it or uninstall the Extension.
  • No Cloud Sync: The Extension currently does not have cloud synchronization features, so we do not have access to any of your data.

2. Permissions Usage

To function correctly, the Extension requests the following permissions, used only when necessary:

  • storage / unlimitedStorage: Used to save your prompt library and preferences on your local device.
  • activeTab / scripting: Used to auto-fill your selected prompts on AI websites you are currently visiting (e.g., ChatGPT, Claude). We interact with the page only when you actively click the extension interface or use a shortcut.
  • contextMenus: Used to add a right-click menu option, allowing you to quickly save selected text as a new prompt.

3. Third-Party Services

The Extension runs purely locally and does not contain any third-party analytics tools (like Google Analytics), advertising SDKs, or tracking code.

4. Data Security

Since your data is stored locally, its security depends on the security of your device. Please avoid saving sensitive information on public devices.

5. Policy Updates

If there are significant changes to our privacy policy, we will notify you via extension update notes or this page. Since we do not collect user information, we cannot notify you via email.

6. Contact Us

If you have any questions about this privacy policy or encounter issues while using the Extension, please contact us via GitHub Issues or the Chrome Web Store support section.

AI Prompt Helper(이하 "본 확장 프로그램")를 이용해 주셔서 감사합니다. 당사는 귀하의 데이터를 어떻게 처리하는지 투명하게 공개하고자 합니다. 요약하자면: 당사는 귀하의 개인정보를 수집하지 않으며, 귀하의 데이터는 전적으로 귀하의 로컬 장치에 저장됩니다.

1. 데이터 수집 및 저장

본 확장 프로그램은 귀하의 개인정보나 사용 데이터를 외부 서버로 수집, 전송 또는 저장하지 않을 것을 약속합니다.

  • 로컬 저장소: 귀하가 생성한 모든 프롬프트와 설정은 브라우저의 chrome.storage.local에 저장됩니다. 이 데이터는 귀하가 직접 내보내거나 확장 프로그램을 삭제하지 않는 한 귀하의 장치에만 유지됩니다.
  • 클라우드 동기화 없음: 본 확장 프로그램은 현재 클라우드 동기화 기능이 없으므로 당사는 귀하의 데이터에 액세스할 수 없습니다.

2. 권한 사용 설명

기능 구현을 위해 본 확장 프로그램은 다음 권한을 요청하며, 필요한 경우에만 사용합니다:

  • storage / unlimitedStorage: 로컬 장치에 프롬프트 라이브러리와 환경 설정을 저장하는 데 사용됩니다.
  • activeTab / scripting: 현재 방문 중인 AI 웹사이트(예: ChatGPT, Claude)에서 선택한 프롬프트를 자동으로 채우는 데 사용됩니다. 귀하가 확장 프로그램 인터페이스를 클릭하거나 단축키를 사용할 때만 페이지와 상호 작용합니다.
  • contextMenus: 웹페이지에 우클릭 메뉴를 추가하여 선택한 텍스트를 새 프롬프트로 빠르게 저장할 수 있도록 하는 데 사용됩니다.

3. 제3자 서비스

본 확장 프로그램은 순수하게 로컬에서 실행되며 제3자 분석 도구(예: Google Analytics), 광고 SDK 또는 추적 코드를 포함하지 않습니다.

4. 데이터 보안

귀하의 데이터는 로컬에 저장되므로 보안은 귀하의 장치 보안에 달려 있습니다. 공용 장치에 민감한 정보를 저장하지 마십시오.

  • Нет облачной синхрониции: В настоящее время в Расширении нет функций облачной синхронизации, поэтому у нас нет доступа к вашим данным.

2. Использование разрешений

Для правильной работы Расширение запрашивает следующие разрешения, используемые только при необходимости:

  • storage / unlimitedStorage: Используется для сохранения вашей библиотеки промптов и настроек на вашем локальном устройстве.
  • activeTab / scripting: Используется для автозаполнения выбранных вами промптов на сайтах ИИ, которые вы посещаете в данный момент (например, ChatGPT, Claude). Мы взаимодействуем со страницей только тогда, когда вы активно нажимаете на интерфейс расширения или используете горячую клавишу.
  • contextMenus: Используется для добавления пункта в контекстное меню, позволяющего быстро сохранить выделенный текст как новый промпт.

3. Сторонние сервисы

Расширение работает исключительно локально и не содержит никаких сторонних инструментов аналитики (таких как Google Analytics), рекламных SDK или кода отслеживания.

4. Безопасность данных

Поскольку ваши данные хранятся локально, их безопасность зависит от безопасности вашего устройства. Пожалуйста, избегайте сохранения конфиденциальной информации на общедоступных устройствах.

AI Prompt Helper (इसके बाद "एक्सटेंशन" के रूप में संदर्भित) में आपका स्वागत है। हम इस बारे में पारदर्शी होना चाहते हैं कि हम आपके डेटा को कैसे संभालते हैं। संक्षेप में: हम आपकी कोई भी व्यक्तिगत जानकारी एकत्र नहीं करते हैं; आपका डेटा पूरी तरह से आपके स्थानीय डिवाइस पर संग्रहीत होता है।

1. डेटा संग्रह और भंडारण

एक्सटेंशन आपकी किसी भी व्यक्तिगत जानकारी या उपयोग डेटा को किसी बाहरी सर्वर पर एकत्र, प्रसारित या संग्रहीत नहीं करने के लिए प्रतिबद्ध है।

  • स्थानीय भंडारण: आपके द्वारा बनाए गए सभी प्रॉम्प्ट और सेटिंग्स आपके ब्राउज़र के chrome.storage.local में संग्रहीत होते हैं। यह डेटा पूरी तरह से आपके डिवाइस पर रहता है जब तक कि आप इसे मैन्युअल रूप से निर्यात नहीं करते या एक्सटेंशन को अनइंस्टॉल नहीं करते।
  • कोई क्लाउड सिंक नहीं: एक्सटेंशन में वर्तमान में क्लाउड सिंक्रोनाइज़ेशन सुविधाएँ नहीं हैं, इसलिए हमारे पास आपके किसी भी डेटा तक पहुँच नहीं है।

2. अनुमतियों का उपयोग

सही ढंग से कार्य करने के लिए, एक्सटेंशन निम्नलिखित अनुमतियों का अनुरोध करता है, जिनका उपयोग केवल आवश्यकता होने पर ही किया जाता है:

  • storage / unlimitedStorage: आपके स्थानीय डिवाइस पर आपकी प्रॉम्प्ट लाइब्रेरी और प्राथमिकताओं को सहेजने के लिए उपयोग किया जाता है।
  • activeTab / scripting: आपके द्वारा वर्तमान में देखी जा रही AI वेबसाइटों (जैसे ChatGPT, Claude) पर आपके चयनित प्रॉम्प्ट को ऑटो-फिल करने के लिए उपयोग किया जाता है। हम पेज के साथ तभी इंटरैक्ट करते हैं जब आप सक्रिय रूप से एक्सटेंशन इंटरफ़ेस पर क्लिक करते हैं या शॉर्टकट का उपयोग करते हैं।
  • contextMenus: एक राइट-क्लिक मेनू विकल्प जोड़ने के लिए उपयोग किया जाता है, जिससे आप चयनित टेक्स्ट को एक नए प्रॉम्प्ट के रूप में तेज़ी से सहेज सकते हैं।

3. तृतीय-पक्ष सेवाएँ

एक्सटेंशन पूरी तरह से स्थानीय रूप से चलता है और इसमें कोई तृतीय-पक्ष विश्लेषण उपकरण (जैसे Google Analytics), विज्ञापन SDK या ट्रैकिंग कोड नहीं है।

4. डेटा सुरक्षा

चूंकि आपका डेटा स्थानीय रूप से संग्रहीत है, इसलिए इसकी सुरक्षा आपके डिवाइस की सुरक्षा पर निर्भर करती है। कृपया सार्वजनिक उपकरणों पर संवेदनशील जानकारी सहेजने से बचें।

5. नीति अपडेट

यदि हमारी गोपनीयता नीति में महत्वपूर्ण परिवर्तन होते हैं, तो हम आपको एक्सटेंशन अपडेट नोट्स या इस पेज के माध्यम से सूचित करेंगे। चूंकि हम उपयोगकर्ता जानकारी एकत्र नहीं करते हैं, इसलिए हम आपको ईमेल के माध्यम से सूचित नहीं कर सकते हैं।

6. हमसे संपर्क करें

यदि आपके पास इस गोपनीयता नीति के बारे में कोई प्रश्न हैं या एक्सटेंशन का उपयोग करते समय समस्याओं का सामना करना पड़ता है, तो कृपया GitHub Issues या Chrome Web Store सहायता अनुभाग के माध्यम से हमसे संपर्क करें।

1. Pengumpulan dan Penyimpanan Data

Ekstensi berkomitmen untuk tidak mengumpulkan, mengirimkan, atau menyimpan informasi pribadi atau data penggunaan Anda di server eksternal mana pun.

  • Penyimpanan Lokal: Semua prompt dan pengaturan yang Anda buat disimpan di chrome.storage.local browser Anda. Data ini tetap berada di perangkat Anda kecuali Anda mengekspornya secara manual atau menghapus instalan Ekstensi.
  • Tidak Ada Sinkronisasi Cloud: Ekstensi saat ini tidak memiliki fitur sinkronisasi cloud, sehingga kami tidak memiliki akses ke data Anda.

2. Penggunaan Izin

Agar berfungsi dengan benar, Ekstensi meminta izin berikut, yang hanya digunakan saat diperlukan:

  • storage / unlimitedStorage: Digunakan untuk menyimpan perpustakaan prompt dan preferensi Anda di perangkat lokal Anda.
  • activeTab / scripting: Digunakan untuk mengisi otomatis prompt pilihan Anda di situs web AI yang sedang Anda kunjungi (misalnya, ChatGPT, Claude). Kami berinteraksi dengan halaman hanya saat Anda mengklik antarmuka ekstensi secara aktif atau menggunakan pintasan.
  • contextMenus: Digunakan untuk menambahkan opsi menu klik kanan, yang memungkinkan Anda menyimpan teks yang dipilih sebagai prompt baru dengan cepat.

3. Layanan Pihak Ketiga

Ekstensi berjalan sepenuhnya secara lokal dan tidak mengandung alat analitik pihak ketiga (seperti Google Analytics), SDK iklan, atau kode pelacakan.

4. Keamanan Data

Karena data Anda disimpan secara lokal, keamanannya bergantung pada keamanan perangkat Anda. Harap hindari menyimpan informasi sensitif di perangkat publik.

5. Pembaruan Kebijakan

Jika ada perubahan signifikan pada kebijakan privasi kami, kami akan memberi tahu Anda melalui catatan pembaruan ekstensi atau halaman ini. Karena kami tidak mengumpulkan informasi pengguna, kami tidak dapat memberi tahu Anda melalui email.

6. Hubungi Kami

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini atau mengalami masalah saat menggunakan Ekstensi, silakan hubungi kami melalui GitHub Issues atau bagian dukungan Chrome Web Store.
